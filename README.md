## ğŸ” Final Project

**Secure File Transfer with RSA and Hashing in Python**

A Python implementation demonstrating:
- RSA key generation (2048-bit)
- File encryption using RSA with OAEP padding
- File decryption
- SHA-256 hashing for file integrity verification
- Secure file transfer simulation

### Key Features:
- **RSA Encryption/Decryption**: Asymmetric encryption for secure file transfer
- **Hash-based Integrity Check**: SHA-256 hashing to verify file integrity
- **Command-line Interface**: Practical implementation with argparse
- **PEM Key Management**: Secure key storage and loading

### Technologies Used:
- Python 3
- `cryptography` library
- `hashlib` for hashing
